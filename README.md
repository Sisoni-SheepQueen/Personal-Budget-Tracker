# Console Personal Budget Tracker

Конзолно C# приложение за управление на лични разходи.
Проектът е разработен като екипна работа с GitHub, branches и Pull Requests.

## Функционалности
- Добавяне на разход (сума, категория, дата)
- Показване на всички разходи
- Изтриване на разход
- Запис и зареждане от файл
- Цветен конзолен интерфейс
- Статистика:
  - Общо похарчени пари
  - Най-голям разход
  - Среден разход
  - Разходи по категории
- Сортиране по:
  - Сума
  - Дата
  - Категория

## Цветове
- Жълто – меню
- Червено – разход над 100 лв
- Зелено – разход под 20 лв

## Работа в екип
- Всеки ученик работи в собствен branch
- Минимум 3 commit-а на човек
- Pull Request + Code Review
- Merge след одобрение

## Технологии
- C#
- .NET Console Application
- GitHub

1. Clone репозитория
